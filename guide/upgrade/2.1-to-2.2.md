## 升级数据库

**预计耗时：** `2 分钟`

- `common_configs` 公用配置

    索引：**删除 `name` 和 `namespace` 的混合 `Promary` 索引**

    | 字段 | 可为空 | 默认值 | 其他 | 描述 | 操作 |
    |-----|-------|------|------|-----|-----|
    | `id` |      |      | `AUTO_INCREMENT` | | 增加 |

    添加索引：

    | 索引字段 | 索引类型  |
    |-----|-----|
    | `id` | `promary` |
    | `name`, `namespace` | `unique` |
    | `name` | `index` |
    | `namespace` | `index` |
